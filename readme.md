This repo holds my configurations and tweaks for Tharkad.

Tharkad is an old(ish) render machine that I'm using as a machine learning inference server. 

Tharkad is currently running llama.cpp and hosts a few models. I'm primarily accessing these models via opencode.

Tharkad specs:
CPU: Threadripper 3960X (24 core)
RAM: 128GB DDR4
GPU: RTX 2080ti (11gb VRAM) x4
